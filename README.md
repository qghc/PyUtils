# PyUtils
some utils of python.

* BaseDao.py-------------------访问MySQL的工具，可以实现ORM存取。
* BaseDao.conf-----------------BaseDao的配置文件
* CommonSpider.py-----------通用的网页爬取器，可以简单配置就能爬取网页信息，支持翻页。
* ImgSpider.py-----------------图片爬取器，可以爬取网页中的特定的图片，支持翻页。
* MatchUtils.py-----------------其他一些文件读取，保存图片之类的小工具。
* TestUtil.py--------------------测试类
